# A real-time collaborative document editor built with React.js, Node.js, Socket.io, and MongoDB. Multiple users can edit the same document simultaneously with changes syncing instantly across all connected clients, and MongoDB for persistent document storage. Users can share a unique document URL, edit content together live, and changes are auto-saved to the database.
